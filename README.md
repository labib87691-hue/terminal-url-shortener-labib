Labib All-In-One CLI Toolkit
A versatile, feature-rich command-line interface (CLI) toolkit written in Python. Designed for developers, system administrators, and tech enthusiasts to perform everyday utility tasks directly from the terminal.

Features
URL Shortener: Instantly convert long links into short TinyURL links.

Weather Update: Fetch real-time weather information for any city worldwide.

IP & Domain Lookup: Discover IP details, geolocation, ISP, and organization info.

System Information: Monitor CPU usage, RAM stats, and system specs.

ASCII Banner Generator: Convert plain text into ASCII text banners.

Tic-Tac-Toe Game: Play a built-in interactive CLI game.

QR Code Generator: Render ASCII QR codes directly inside your terminal screen.

GitHub User Analyzer: Inspect GitHub user profiles, repos, and follower stats.

Disk Usage Visualizer: View drive partition usage with a visual progress bar.

Prerequisites & Installation
Make sure you have Python 3.x installed on your system.

1. Clone the Repository
git clone https://github.com/YOUR-USERNAME/terminal-url-shortener-labib.git
cd terminal-url-shortener-labib

2. Install Required Dependencies
Run the following command to install all necessary Python packages:

pip install requests psutil pyfiglet qrcode

How to Run
Execute the script using Python in your Command Prompt, PowerShell, or Linux Terminal:

python toolkit.py
